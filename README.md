# front-end-code-test

A front end  code test I did as part of a job application process for Mdev Digital. It involved building a front end component from a mockup and stye guide created in Zeplin. Makes use of css grid.

## Project setup
```
npm install
```
### Start project with
```
npm run start
```

### Or
```
npm install gulp -g
gulp watch
```
